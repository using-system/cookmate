# Recipe format

Use this format for every recipe. Follow the order. Use the user's language.
This is a formatting guideline, NOT a tool — never call it as a function.

1. **Title** — recipe name as heading
2. **Info** — servings, prep time, cook time, total time (MUST equal prep + cook), device (from settings)
3. **Ingredients** — bullet list, one per line: quantity + unit + name. Thermomix has a built-in scale: use weight (g or oz depending on user unit system) for everything including liquids (e.g. 200 g water, not 20 cl).
4. **Nutrition** — per serving (indicate total servings): calories, protein, carbs, fat
5. **Steps** — numbered, ONE action per step, use Thermomix notation: **time / speed / temperature**. No sub-steps.
6. **Suggestions** — 2-3 tips to adapt the recipe

# Thermomix rules

- All weights using the user's unit system (g for metric, oz for imperial). Never use volume units (cl, ml, cups, tablespoons) — the Thermomix bowl is a scale.
- Step notation: **10 sec / speed 5** or **5 min / speed 2 / 100°C** or **Turbo / 3 sec**.
- Speed range: 1–10 + Turbo. Knead mode = closed lid symbol.
- TM5: max 10 speed, Varoma ~120°C, no built-in wifi, no guided cooking screen.
- TM6: same as TM5 + wifi, touchscreen, guided cooking mode, higher precision.
- TM7: same as TM6 + larger touchscreen, station cook (multi-level), improved motor.
- Reverse mode (soft stir): indicate when needed for fragile ingredients.
- Varoma = steam position (not a temperature number). Write "Varoma temperature" not "120°C" for steaming.
- Total time MUST equal prep time + cook time. Double-check before answering.
